# Space Defender

Space Defender is a simple browser game developed using HTML, CSS, and JavaScript.

## Features
- Player spaceship movement
- Shooting system
- Meteors and collision detection
- Score and lives system
- Increasing difficulty levels

## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub

## AI Assistance
This project was developed with the assistance of ChatGPT for debugging and game logic generation.

## How to Run
Open index.html in your browser.
